# MedTransfer Pro 🏥
## 医学部学士編入試験 完全対策プラットフォーム

---

## 📌 プロジェクト概要

MedTransfer Pro は、医学部学士編入試験を目指す受験生のための高品質な対策Webアプリです。
3万円の買い切りプレミアムモデルを採用し、試験情報の「情報戦」を勝ち抜くための完全データベースを提供します。

**デザイン方針:** ネイビー × ゴールド × ベージュの信頼性の高いカラーパレット

---

## ✅ 現在実装済みの機能

### タブ1: 🏫 募集要項データベース
- 全国 **27大学** の医学部学士編入試験情報を一覧表示
- 地方フィルター（北海道・東北 / 関東 / 中部 / 近畿 / 中国・四国 / 九州）
- 生命科学重要度フィルター（◎重点 / ◎・○ / 全て）
- テキスト検索（大学名・特徴文でリアルタイム検索）
- ◎/○/△/× のカラーバッジ表示（ネイビー/ゴールド/グレー/薄グレー）
- クリックで詳細モーダル（配点・試験特徴）

### タブ2: 📝 過去問アーカイブ（ペイウォール付き）
- **KALS 生命科学 基礎シリーズ 38問**を収録（QUIZ 24問 / DRILL 6問 / EXERCISE 8問）
- 章別フィルター（第1講 / 第2〜4講 / 第5講）
- 種別フィルター（QUIZ / DRILL / EXERCISE）
- **ペイウォール機能:**
  - 1問目は無料で完全閲覧可能
  - 2問目以降はロック（ブラーオーバーレイ + ロックアイコン）
  - クリックでプレミアム誘導モーダル表示
- 解答折りたたみ表示（「解答を表示」ボタン）
- 正解/不正解マーク機能（ローカルストレージに保存）
- デモモード解放ボタン（テスト用）

### タブ3: 📅 受験スケジュール
- **10大学**の出願・1次・2次試験日程を収録（2026年度対応）
- カレンダーグリッドビュー（月別、色分けイベント）
- タイムラインリストビュー（日付順ソート）
- 出願(赤) / 1次試験(青) / 2次試験(紫) の色分け
- 月ナビゲーション（4〜9月対応）

### タブ4: 📊 学習ダッシュボード
- 正解数 / 不正解数 / 正答率 / 連続学習日ストリーク
- 章別正答率グラフ（Chart.js 棒グラフ）
- 種別分布チャート（Chart.js ドーナツグラフ）
- 章別進捗バー（アニメーション付き）
- 弱点問題リスト（不正解問題を自動抽出）
- 学習データリセット機能
- JSON形式エクスポート機能

### タブ5: ⚙️ 設定/アカウント
- アカウント情報表示（デモ）
- プレミアムステータス表示（無料/プレミアム）
- テーマ切り替え（ライト/ダークモード）
- 技術構成（Tech Stack）表示
- セキュリティ対策説明（署名付きURL, DevTools対策等）

### モーダル
- **プレミアムアップグレードモーダル:** 価格表示、機能一覧、Stripe決済ボタン
- **Stripe Checkoutデモモーダル:** カード情報入力フォーム（デモ）
- **大学詳細モーダル:** 大学別の詳細情報
- **決済完了演出:** 購入完了アニメーション + 全コンテンツ解放

## 🔒 セキュリティ設計

- **署名付きURL（Signed URLs）**: PDF閲覧URLは発行から60秒で無効化
- **直接ダウンロード防止**: Storage pathはサーバーAPIのみ保有
- **ユーザー権限チェック**: 全APIコールでSupabase Auth経由の認証確認
- **DevTools対策**: フロントエンドからのURL取得不可
